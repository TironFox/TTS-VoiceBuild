# 项目代码审查与缺失遗漏修复计划

## 审查范围
- 前端：`src/App.tsx`, `src/components/`, `src/types.ts`, `src/main.tsx`, `src/index.css`
- 后端：`server.ts`
- 配置：`package.json`, `vite.config.ts`, `tsconfig.json`, `index.html`, `.env.example`

---

## 一、发现的问题与修复计划

### 问题 1：index.html 标题和语言设置不正确
**文件**: `index.html`
**问题**:
- 标题为 "My Google AI Studio App"，未反映项目实际功能
- 语言设置为 `lang="en"`，应为 `lang="zh-CN"`

**修复**:
```html
<html lang="zh-CN">
  <title>阿里百炼音色设计与分段可控语音合成控制台</title>
```

---

### 问题 2：TextSegment emotion 类型不完整
**文件**: `src/types.ts` vs `src/App.tsx`
**问题**:
- `types.ts` 定义: `'neutral' | 'happy' | 'sad' | 'angry' | 'fear' | 'excited' | 'serious' | 'gentle' | 'professional'` (9种)
- `App.tsx` select 组件只实现了部分选项，缺少 `angry` 和 `fear`

**修复**:
在 `App.tsx` 第 1951-1957 行的 emotion select 中添加缺失选项：
```tsx
<option value="angry">愤怒</option>
<option value="fear">恐惧</option>
```

---

### 问题 3：克隆 API 返回字段不一致
**文件**: `server.ts` vs `App.tsx`
**问题**:
- `server.ts` 第 662 行返回: `voiceId: data.output?.voice_id || ...`
- `App.tsx` 第 405-420 行使用 `data.voiceId`
- 字段名存在不一致风险

**修复**:
确认 `server.ts` 返回的字段名为 `voiceId`，保持前端引用不变。

---

```typescript
// 第 203 行
const recordingTimerRef = useRef<number | null>(null);

```

---

### 问题 5：生产构建脚本不完整
**文件**: `package.json`
**问题**:
- `build` 脚本没有处理 Tailwind CSS 4 的 JIT 编译
- Vite 生产构建后直接打包 server.ts，缺少正确的产品环境配置

**修复**:
更新 `package.json` 的 build 脚本：
```json
"build": "vite build",
"server:build": "esbuild server.ts --bundle --platform=node --format=cjs --packages=external --sourcemap --outfile=dist/server.cjs",
```

---

### 问题 6：setSegments 状态更新错误
**文件**: `src/App.tsx` 第 1001 行
**问题**:
```typescript
setSegments([...segments]); // 这不会触发重新渲染因为引用相同
```
在异步递归函数 `playStep` 中直接修改 segments 副本并设置状态，引用未变不会触发更新。

**修复**:
```typescript
setSegments(prev => prev.map(s =>
  s.id === activeSeg.id
    ? { ...s, audioUrl: data.audioData }
    : s
));
```

---

### 问题 7：缺少全局错误边界
**文件**: `src/main.tsx`
**问题**:
- 没有 React Error Boundary 组件
- API 请求失败时可能导致整个应用崩溃

**修复**:
创建 `ErrorBoundary.tsx` 组件并包裹 `<App />`。

---

### 问题 8：vite.config.ts 未配置 HTTPS
**文件**: `vite.config.ts`
**问题**:
- 开发环境没有配置 HTTPS
- 浏览器麦克风权限在非安全上下文中可能受限

**修复** (可选):
如需 HTTPS 开发，添加 `server.https` 配置。

---

### 问题 9：.gitignore 缺少 uploads 目录
**文件**: `.gitignore`
**问题**:
- `uploads/` 目录（server.ts 第 24 行创建）未在 gitignore 中
- 可能提交用户上传的临时音频文件

**修复**:
```gitignore
uploads/
```

---

### 问题 10：README.md 内容未更新
**文件**: `README.md`
**问题**:
- 内容仍为 Google AI Studio 默认模板
- 没有项目实际使用说明

**修复**:
更新 README.md 描述项目的实际功能、依赖和运行方法。
